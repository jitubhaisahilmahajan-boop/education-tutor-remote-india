# The Education Tutor for Remote India

An AI-powered education platform leveraging **Context Pruning** to deliver personalized, bandwidth-efficient learning for remote and rural Indian students.

## 🎯 Problem Statement

Millions of students in remote/rural India have limited access to quality education and learning resources. This project aims to bridge that gap using AI-driven personalized tutoring with optimized content delivery.

## ✨ Key Features

- **Personalized Learning**: Adaptive content based on student progress and learning style
- **Context Pruning**: Reduces bandwidth by delivering only relevant educational content
- **Multi-language Support**: Vernacular language support for better accessibility
- **Low-Bandwidth Ready**: Works in areas with limited or intermittent connectivity
- **Analytics Dashboard**: Track student progress and learning patterns
- **Scalable Architecture**: Cloud-ready for serving thousands of students
